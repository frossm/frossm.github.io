RPNCalc: The Command Line RPN Calculator

Trigonometry Functions

CommandDescription
sin, cos, tan [rad]Calculate the trigonometry function. Angles are input as degrees by default unless the rad parameter is given in which case the angles will be in radians. Example: tan will calculate the tangent using row 1 as the angle in degrees. Use tan rad if row 1 contains the angle in radians
asin, acos, atan [rad]Calculate the arc trigonometry function. Result is returned in degrees unless rad parameter is provided
hypot
hypotenuse		Assumes the top two stack items are the right triangles legs and this function returns the hypotenuse using the Pythagorean theorem. Specifically, it returns SQRT( (line1)^2 + (line2)^2 )
